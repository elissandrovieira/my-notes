- It are steps that must be followed when a software product is developed.
- Process Model

## Three generical phases

- Definition
- Development
- Maintenance
---
## Definition

The steps are:
- Requirements engineering
	- Client meetings
	- Functional requirements
		- Must-haves features
	- Non-functional requirements
		- Forbidden features
- Problem Definition
- Systems Analysis
	- This step is completely linked with the paradigm used
	- Problem division
	- Relationship between subproblems

## Development

The steps are:
 - Programming Language
	 - Coding
		 - C, C#, Java, Php
- Algorithm
- Tests
	- Quality patterns

## Maintenance

The steps are:
- Manual
- Modification
- Problem solving
- User training
